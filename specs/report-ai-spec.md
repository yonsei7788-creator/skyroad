## 1. 설계 원칙

### 1.1 예시 분석본에서 도출한 핵심 구조

4개의 예시 분석본을 분석한 결과, 공통적으로 다음 구조를 따른다:

1. **학생 유형 프로파일링 + 레이더 차트**: 학생의 핵심 특성을 직관적으로 보여주는 유형 분류와 4대 역량 시각화
2. **정량 스코어링 + 합격 예측**: 300점 만점의 역량 점수와 전형별 합격 가능성을 수치화
3. **과목별 원문 인용 + 상세 평가**: 생기부 원문을 테이블 좌측에, 평가를 우측에 배치. 평가에는 학업역량/진로역량/공동체역량 관련 태깅이 포함됨
4. **종합 평가**: 장점/단점 테이블 + 4대 역량(학업/진로/공동체/발전가능성)별 총평
5. **성적 분석**: 수강자수, 표준편차, 등급 분포, 과목 간 편차, 전공 관련 교과 이수 노력, 전공 관련 교과 성취도를 다차원으로 분석
6. **전략적 추천**: 교과/학종/정시 전형별 전략, 주의해야 할 학교 유형, 유리한 학교 유형
7. **대학 추천**: 입시 데이터(50%/70% 컷, 경쟁률, 모집인원) 기반 구체적 대학 추천
8. **세특 주제 추천**: 과목별 맞춤형 후속 탐구 주제 제안
9. **3학년 전략/스토리라인**: 앞으로의 생기부 방향 설계

### 1.2 예시 대비 개선 방향

예시 분석본은 전문 컨설턴트가 작성한 것으로 고품질이지만, AI 리포트는 다음을 개선한다:

| 영역           | 예시 분석본              | AI 리포트 개선점                                                                |
| -------------- | ------------------------ | ------------------------------------------------------------------------------- |
| 학생 프로파일  | 없음                     | **유형 분류 + 레이더 차트** (학생 유형명, 캐치프레이즈, 태그로 직관적 파악)     |
| 역량 평가 체계 | 암묵적으로 3대 역량 언급 | **300점 만점 정량 스코어링** (학업100+진로100+공동체100 + 발전가능성 별도 등급) |
| 합격 예측      | 전형 추천 수준           | **전형별 합격률 수치화** (학종/교과/정시 각각 확률 범위 제시)                   |
| 성적 분석      | 텍스트 기반 해석         | **정량적 다차원 분석** (Z-score, 백분위, 등급 환산 테이블, 5등급제 시뮬레이션)  |
| 과목 간 연결   | 간헐적 언급              | **체계적 스토리라인 분석** (과목 간 주제 연결 그래프)                           |
| 대학 추천      | 3~6개 대학 나열          | **전형별 전략 매트릭스** (상향/안정/하향 x 학종/교과/정시)                      |
| 문장 단위 분석 | 일부 문장에 코멘트       | **Premium: 전 문장 체계적 분석** (역량 태깅 + 개선 제안)                        |
| 기록 충실도    | 없음                     | **기록 분량 대비 질 분석** (최대 기재 분량 대비 실제 기록량 + 질 평가)          |
| 추천 도서/전공 | 없음                     | **AI 기반 도서 추천 + 전공 추천** (학생 기록과의 연결성 분석)                   |

### 1.3 핵심 설계 결정

1. **3파트 + 부록 구조** -- Part 1 진단, Part 2 분석, Part 3 전략, 부록으로 21섹션 구성
2. **4대 역량 프레임워크를 평가의 뼈대로 사용** -- 대학의 실제 평가 축(학업역량/진로역량/공동체역량/발전가능성)을 리포트 전체에 관통
3. **300점 만점 정량 스코어** -- 학업100 + 진로100 + 공동체100, 발전가능성은 별도 등급
4. **Premium만 문장 단위 세특 분석 제공**
5. **교과 세특과 창체 분석 분리** -- subjectAnalysis(교과 전용) + activityAnalysis(창체 전용)
6. **고1 전용 directionGuide** -- admissionStrategy 대체, 방향 설정 가이드 제공

## 2. 4대 평가 역량 프레임워크

예시 분석본과 평가 가이드에서 도출한 평가 축. 대학의 실제 4가지 평가 축(학업역량/진로역량/공동체역량/발전가능성)을 리포트 전체에 관통한다.

> **설계 근거**: `require/ai-report-evaluation-guide.md`에 명시된 대학의 4가지 평가 축(전공적합성, 학업역량, 탐구역량, 발전가능성)을 리포트 프레임워크에 반영. 전공적합성과 탐구역량은 각각 진로역량과 학업역량에 통합하되, **발전가능성은 독립 축으로 분리**하여 평가의 완전성을 보장한다.

### 2.1 학업역량 (academic)

| 하위 항목      | 평가 기준                                                            | 입력 데이터      |
| -------------- | -------------------------------------------------------------------- | ---------------- |
| **학업성취도** | 내신 등급, 등급 추이, 과목 간 편차, 수강자수/표준편차 대비 실질 위치 | 교과학습발달상황 |
| **학업태도**   | 수업 참여도, 자기주도적 학습, 성실성 (세특 서술에서 추출)            | 세특, 행동특성   |
| **탐구력**     | 지적 호기심, 탐구의 깊이/확장성, 꼬리물기 탐구 여부, 과정 중심 서술  | 세특, 창체       |

### 2.2 진로역량 (career)

| 하위 항목                    | 평가 기준                                                                                                                                                                                                                                                                                                     | 입력 데이터                    |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------ |
| **전공 관련 교과 이수 노력** | 핵심/일반 권장과목 이수 여부, 공동교육과정 이수, 진로선택과목 선택. **2028학년도 계열별 권장 선택과목 이수 여부를 기반으로 평가**: 메디컬 계열은 대학별 필수/권장 수학/과학 과목 이수 확인, 일반 계열은 핵심 교과 + 권장 선택과목 이수 여부 확인. 참조 데이터: `libs/report/constants/recommended-courses.ts` | 교과학습발달상황, 진로선택과목 |
| **전공 관련 교과 성취도**    | 전공 관련 과목의 성적이 전체 대비 어떤지                                                                                                                                                                                                                                                                      | 교과학습발달상황               |
| **진로 탐색 활동 및 경험**   | 진로 방향 일관성, 학년별 심화도, 진로 관련 구체적 활동                                                                                                                                                                                                                                                        | 창체(진로활동), 세특, 독서     |

### 2.3 공동체역량 (community)

| 하위 항목             | 평가 기준                                    | 입력 데이터              |
| --------------------- | -------------------------------------------- | ------------------------ |
| **나눔과 배려**       | 친구 학습 도움, 타인 의견 존중, 배려 사례    | 세특, 행동특성, 봉사     |
| **소통 및 협업**      | 모둠 활동, 토론 참여, 발표 역량, 팀 프로젝트 | 세특, 창체               |
| **리더십**            | 임원 활동, 모둠장/부장 역할, 주도적 기획     | 창체(자율활동), 세특     |
| **성실성과 규칙준수** | 출결, 역할 이행, 과제 수행 성실도            | 출결상황, 행동특성, 세특 |

### 2.4 발전가능성 (growth)

| 하위 항목           | 평가 기준                                                                  | 입력 데이터                 |
| ------------------- | -------------------------------------------------------------------------- | --------------------------- |
| **자기주도성**      | 스스로 질문을 만들고 학습을 확장하는 경험, 교사 지도 없이도 추가 탐구 수행 | 세특, 창체, 행동특성        |
| **경험의 다양성**   | 다양한 영역의 활동 경험, 교과 내/외 경험의 폭                              | 창체, 세특, 독서            |
| **성장 과정**       | 학년별 활동의 점진적 심화, 시행착오와 한계 인식, 보완 시도의 흔적          | 세특(학년별 비교), 행동특성 |
| **창의적 문제해결** | 기존과 다른 접근, 독창적 아이디어, 문제 정의-해결-확장의 사고 흐름         | 세특, 창체                  |

> **발전가능성의 핵심 질문**: "이 학생이 대학에 와서도 계속 성장할 수 있는가?"
>
> - 성공 경험뿐 아니라 시행착오, 한계 인식, 보완 시도가 오히려 긍정적
> - 활동들이 시간 흐름에 따라 점점 심화되고 구체화되는지 평가

### 3.3 각 섹션 상세 명세

---

#### 섹션 1: 학생 프로필 (studentProfile)

**파트**: Part 1 진단
**목적**: 학생의 핵심 특성을 유형화하여 직관적으로 파악할 수 있는 프로필 제공.

**출력 구성**:

| 항목                                            | Lite | Standard | Premium |
| ----------------------------------------------- | ---- | -------- | ------- |
| 유형명 (typeName)                               | O    | O        | O       |
| 유형 설명 (typeDescription, 2~3줄)              | O    | O        | O       |
| 레이더 차트 데이터 (radarChart, 4대 역량 0~100) | O    | O        | O       |
| 핵심 키워드 태그 (tags, 3~5개)                  | O    | O        | O       |
| 한줄 캐치프레이즈 (catchPhrase)                 | O    | O        | O       |

**유형명 예시**:

- "탐구형 성장러" -- 꼬리물기 탐구와 학년별 심화가 돋보이는 학생
- "실행형 리더" -- 주도적 기획과 팀 프로젝트 경험이 풍부한 학생
- "균형형 올라운더" -- 학업/진로/공동체 역량이 고르게 발달한 학생
- "진로집중형 탐색가" -- 특정 진로에 대한 깊은 탐구와 일관된 방향성

**레이더 차트 데이터**:

- `academic` (학업역량): 0~100
- `career` (진로역량): 0~100
- `community` (공동체역량): 0~100
- `growth` (발전가능성): 0~100

**태그 예시**: `["진로 일관성 우수", "탐구 깊이 양호", "리더십 돋보임"]`

**캐치프레이즈 예시**: "자동차에서 시작해 모빌리티의 미래를 설계하는 공학도"

**입력 데이터**: 전체 생기부 데이터 (모든 섹션 종합)

---

#### 섹션 2: 역량 정량 스코어 (competencyScore)

**파트**: Part 1 진단
**목적**: 4대 역량을 300점 만점으로 정량화하고, 발전가능성은 별도 등급으로 표시.

**스코어링 체계**:

| 역량                   | 배점      | 하위항목 (예시 배분)                                  |
| ---------------------- | --------- | ----------------------------------------------------- |
| 학업역량 (academic)    | 100점     | 학업성취도 40 + 학업태도 30 + 탐구력 30               |
| 진로역량 (career)      | 100점     | 교과이수노력 30 + 교과성취도 30 + 진로탐색 40         |

#### 섹션 6: 성적 분석 (academicAnalysis)

**목적**: 교과 성적을 다차원으로 분석하여 학생의 실질적 학업 위치를 파악.

##### Lite: 기본 성적 분석

| 항목                                        | 내용                                                     |

##### Standard: 다차원 성적 분석

| 항목                                                       | 내용                                   |
| ---------------------------------------------------------- | -------------------------------------- |
| 원점수-평균-표준편차 분석 (subjectStatAnalyses)            | 과목별 Z-score, 백분위 추정, 해석      |
| 과목 간 편차 분석 (gradeDeviationAnalysis)                 | 최고/최저 과목, 편차 범위, 리스크 평가 |

| 고교 유형별 환산 (schoolTypeAdjustment)                    | 일반고/특목고/자사고/특성화고 환산     |
| 등급 변화 가능성 (gradeChangeAnalysis)                     | 현재 추이 기반 향후 예측 + 액션 아이템 |
| 진로선택과목 분석 (careerSubjectAnalyses)                  | 성취도별 분포비율 기반 실질 위치       |
| 소인수 과목 분석 (smallClassSubjectAnalyses)               | 수강자 5명 이하 과목 별도 해석         |
| 성적 인플레이션 맥락 (gradeInflationContext)               | 5등급제 하 1등급 비중 확대 맥락 안내   |

##### Premium: 다차원 + 전략적 성적 분석

Standard의 모든 항목 + 추가:

| 항목                                                     | 내용                                       |
| -------------------------------------------------------- | ------------------------------------------ |
| 5등급제 전환 시뮬레이션 (fiveGradeSimulation)            | 현 성적이 5등급제로 전환 시 위치 분석      |
| 대학별 반영 방법 시뮬레이션 (universityGradeSimulations) | 추천 대학들의 성적 반영 방법에 따른 유불리 |
| 성적 개선 우선순위 (improvementPriority)                 | 등급 변화가 가장 효과적인 과목 식별        |

**성적 해석 기준** (평가 가이드 기반):

| 지표 조합                   | 해석                                             |
| --------------------------- | ------------------------------------------------ |
| 표준편차 크고 + 원점수 높음 | 어려운 시험에서 우수한 성적 -> 높은 평가         |
| 표준편차 작고 + 원점수 높음 | 쉬운 시험 -> 높은 등급이어도 변별력 낮음         |
| 수강자수 많고 + 등급 높음   | 큰 경쟁 집단에서의 성과 -> 높은 평가             |
| 수강자수 적고 + 등급 높음   | 경쟁 구조 제한 -> 전공 연관성과 함께 맥락적 해석 |

**소인수 과목 기준**:

- 2015 개정 교육과정: 수강자 13명 이하
- 2022 개정 교육과정: 수강자 5명 이하
- 석차등급 `·` 표기 과목은 성취도 + 원점수 + 과목평균 + 표준편차 복합 평가

**입력 데이터**: 교과학습발달상황 (일반선택/진로선택/체육예술), 학생 프로필 (학교 유형, 목표 전공)

#### 섹션 10: 교과 세특 분석 (subjectAnalysis)

**파트**: Part 2 분석
**목적**: 교과 세특을 과목별로 분석하여 평가 포인트와 개선 방향 제시.

> **중요**: 이 섹션은 **교과 세특만** 분석한다. 창체는 섹션 9(activityAnalysis)에서 분석한다.

이 섹션은 플랜별 차이가 가장 큰 핵심 섹션이다.

##### Lite: 기본 세특 분석

각 과목에 대해:

| 항목                             | 내용                                                                          |
| -------------------------------- | ----------------------------------------------------------------------------- |
| 과목명 (subjectName)             | 예: "수학1", "물리학1" 등                                                     |
| 학년 (year)                      | 해당 학년                                                                     |
| 평가 등급 (rating)               | `excellent` / `good` / `average` / `weak`                                     |
| 역량 태깅 (competencyTags)       | 해당 세특에서 드러나는 역량 태그 (예: `학업역량-탐구력`, `공동체역량-리더십`) |
| 핵심 활동 요약 (activitySummary) | 주요 활동 1~2개 요약 (2~3줄)                                                  |
| 평가 코멘트 (evaluationComment)  | 강점/약점 간략 서술 (3~4줄)                                                   |

##### Standard: 상세 세특 분석

Lite의 모든 항목 + 추가:

| 항목                                     | 내용                                          |
| ---------------------------------------- | --------------------------------------------- |
| 원문 핵심 인용 (keyQuotes)               | 세특에서 평가에 중요한 문장 2~3개 발췌 인용   |
| 상세 평가 (detailedEvaluation)           | 예시 분석본 수준의 구체적 평가 (1문단, 5~8줄) |
| 개선 방향 (improvementDirection)         | 3학년 세특에서 보완할 구체적 방향 (2~3줄)     |
| 개선 예시 문장 (improvementExample)      | 약한 부분을 어떻게 개선했으면 좋았을지 예시   |
| 교과 간 연결성 (crossSubjectConnections) | 다른 과목 세특과의 주제 연결/중복 여부 분석   |

**교과 간 연결성 타입**:

- `주제연결`: 같은 주제를 다른 과목에서 심화하여 탐구
- `역량연결`: 다른 과목에서 동일 역량이 반복 확인
- `중복`: 같은 내용이 단순 반복되어 깊이 추가 없음

##### Premium: 문장 단위 정밀 분석

Standard의 모든 항목 + 추가:

| 항목                              | 내용                                                          |
| --------------------------------- | ------------------------------------------------------------- |
| 문장 단위 분석 (sentenceAnalysis) | 세특 원문의 각 문장(또는 문장 그룹)에 대한 개별 평가          |
| 문장별 역량 태깅                  | 각 문장이 어필하는 역량 태그                                  |
| 하이라이트 표시 (highlight)       | positive(인상적) / negative(감점) / neutral(중립)             |
| 개선 제안 (improvementSuggestion) | 약한 문장에 대한 구체적 개선 예시                             |
| 중요도 퍼센트 (importancePercent) | 해당 과목 세특이 전체 생기부에서 차지하는 평가 중요도 (0~100) |
| 평가 영향도 (evaluationImpact)    | `high` / `medium` / `low` -- 합격 판단에 미치는 영향          |

**문장 단위 분석 예시** (Premium):

```
원문: "에너지를 생산하기 위해 소비되는 비효율적 구조를 고찰하며, 식물을 활용한 에너지 생산의 가능성을 심도 있게 탐구함."

평가: "심도 있게 탐구했다"는 서술이 있으나 구체적으로 어떤 내용을 탐구했는지 드러나지 않음.
      예를 들어 "인공 광합성 시스템과 식물-미생물 연료 전지를 중심으로" 같은 구체적 키워드가
      포함되었다면 탐구 깊이를 평가자가 파악할 수 있었을 것.

역량: 학업역량-탐구력 (미흡)
하이라이트: negative
개선 제안: "광합성 시스템의 에너지 변환 효율(현재 ~1%)의 한계를 인식하고,
           인공 광합성과 식물-미생물 연료전지를 비교 분석하여
           열역학적 관점에서 효율 개선 방향을 제시함"
```

**입력 데이터**: 세부능력 및 특기사항, 학생 프로필 (학년, 목표 전공)

#### 섹션 14: 세특 주제 추천 (topicRecommendation)

| 항목                              | Lite                          | Standard                                        | Premium                                         |

| 추천 주제 수                      | 3개                           | 5개                                             | 5개                                             |
| 주제 구성                         | 주제명 + 연계 과목 + 1줄 설명 | 주제명 + 연계 과목 + 선정 이유 + 기존 탐구 연결 | 주제명 + 연계 과목 + 선정 이유 + 기존 탐구 연결 |

| 세특 서술 예시 (sampleEvaluation) | X                             | X                                               | O                                               |

**주제 추천 기준**:

1. 학생의 기존 탐구 흐름과 자연스럽게 이어지는 주제
2. 부족한 역량을 보완할 수 있는 주제
3. 전공 관련성이 높은 주제
4. 탐구 깊이를 보여줄 수 있는 주제 (단순 조사가 아닌, 실험/분석/비교/모델링 등)

**추천 예시** (Premium):

```
주제: ATP 생산 효율을 떨어뜨리는 요인 분석
연계 과목: 생명과학1
선정 이유: 기존 보고서에서 "린 생산 방식"과 ATP를 연결했으나 생명과학적 깊이 부족.
           이 주제로 ATP의 생물학적 원리를 깊이 탐구하면 학업역량-탐구력 보완 가능.
기존 탐구 연결: 2학년 기술가정 "린 생산 방식과 에너지 효율" 탐구의 심화

활동 설계:
  1단계: ATP 합성 과정 (해당 과정/TCA 회로/전자전달계) 학습
  2단계: 각 과정에서 효율 저하 요인 문헌 조사
  3단계: 미토콘드리아 구조 이상과 ATP 생산 감소의 관계 분석
  4단계: 실생활 사례 연결 (운동 피로, 대사 질환 등)
  소요 기간: 2~3주
  예상 결과물: 탐구 보고서 (A4 3~5매)

세특 서술 예시:
  "세포 에너지 대사를 학습한 뒤 ATP 합성 효율에 영향을 미치는 요인에
  관심을 가지고 탐구를 수행함. 전자전달계에서 양성자 기울기가 손실되는
  기전(uncoupling protein, 열 발생)을 분석하고, 이를 갈색지방조직의
  체온 조절 메커니즘과 연결하여 에너지 대사의 진화적 의미를 고찰함."
```

**입력 데이터**: 세특, 학생 프로필 (목표 학과, 학년), 교과학습발달상황

**제공 조건**: 고2 이상 생기부가 포함된 경우에만 제공. 고1 생기부만 있는 경우 `directionGuide`로 대체.

##### Lite: 기본 입시 전략

| 항목                             | 내용                                                         |
| -------------------------------- | ------------------------------------------------------------ |
| 추천 전형 방향 (recommendedPath) | 학종/교과/정시 중 추천 전형 + 이유 (3~5줄)                   |

| 항목                                          | 내용                                                       |
| --------------------------------------------- | ---------------------------------------------------------- |
| 전형별 상세 전략 (typeStrategies)             | 학종/교과/정시 각각 분석 + 적합도(적합/보통/부적합) + 이유 |

Standard의 모든 항목 + 추가:

| 항목                                                       | 내용                                                        |
| ---------------------------------------------------------- | ----------------------------------------------------------- |
| 수능 최저 전략 (csatMinimumStrategy)                       | 수능 최저 배치 전략 (교과 전형 안정 + 학종 상향 조합)       |
| 지원 조합 시뮬레이션 (applicationSimulation)               | 6장 카드 배분 전략 예시 (교과 2장 + 학종 3장 + 정시 1장 등) |
| 대학별 학종 가이드북 키워드 매칭 (universityGuideMatching) | 각 대학이 강조하는 평가 요소와 학생 강약점 매칭             |

**대학 추천 기준** (대학 등급 매핑 테이블 활용):

1. 학생의 환산 등급 (고교 유형별 환산 적용) 기준 대학 라인 산정
2. 상향: 환산 등급 대비 0.2~0.3 위 라인
3. 안정: 환산 등급과 일치하는 라인
4. 하향: 환산 등급 대비 0.2~0.3 아래 라인
5. 목표 학과의 입시 데이터 반영

**주의/유리 학교 유형 판단 기준** (예시 분석본에서 도출):

| 학생 특성            | 피해야 할 학교              | 유리한 학교               |
| -------------------- | --------------------------- | ------------------------- |
| 과목 간 성적 편차 큼 | 고른 성적 분포 중시 학교    | 전공 과목 성적 위주 학교  |

| 공동교육과정 이수    | --                          | 교과 이수 노력 중시 학교  |
| 출결 이슈            | 출결 엄격 반영 학교         | --                        |

**정시 학생부 반영 확대 안내**:

- 상위 15개 대학 중 절반 이상이 정시에 학생부를 반영하는 추세

**학교폭력 주의사항**:

**입력 데이터**: 교과학습발달상황, 학생 프로필 (목표 대학/학과, 학교 유형), 대학 등급 매핑 테이블

각 연결에 대해 `from`(출발 과목) -> `to`(도착 과목) + `topic`(연결 주제) + `depth`(심화/반복/확장/무관) 표시.

**스토리라인 분석 예시**:

```
메인 스토리라인:
"철학적 사유를 바탕으로 사회 문제를 분석하고 해결하고자 하는 학생"

학년별 흐름:
- 1학년: 철학/경영/법 등 다방면 관심 탐색기 (테마: "지적 호기심의 확산")
- 2학년: 철학과로 진로 구체화, 안락사/정의/실존주의 등 윤리적 주제 탐구 (테마: "윤리적 문제의식")
- 3학년 (제안): 특정 철학 분과(예: 법철학)에 집중하여 심화 탐구

과목 간 연결 (Premium):
- 화법과작문(안락사 비평문) -> 동아리(안락사 윤리 토론) -> 자율(안락사 토론회)
  = 주제 반복이나 심화 부족 (depth: 반복)
- 한국지리(풍수지리+하이데거) -> 철학(실존주의+자본주의)
  = 철학적 분석 역량의 학년 간 심화 확인 (depth: 심화)

진로 일관성: B등급
사유: 1학년 때 경영/법/철학 혼재, 2학년부터 철학 집중.
      1학년의 탐색은 자연스러우나, 법학 관련 활동이 부족하여 "변호사" 진로와의 연결이 약함.

스토리 강화 제안 (Premium):
- 3학년 윤리와사상 세특에서 '법철학'을 중심 주제로 설정하여 진로와 학문의 연결을 완성
- 정치와법 과목에서 실제 판례 분석을 통해 법철학 탐구를 실증적으로 확장

면접 스토리텔링 가이드 (Premium):
"1학년 때 다양한 사회 현상에 관심을 가지다가 -> 2학년에서 윤리적 딜레마에 주목하게 됐고 ->
특히 안락사 문제를 탐구하면서 법과 윤리의 경계에 관심이 생겨 -> 법철학을 전공하고 싶다는
방향을 잡았다"는 흐름으로 설명하면 자연스러운 성장 스토리가 됩니다.
```

**입력 데이터**: 전체 세특, 창체, 행동특성, 독서활동

## 6. 평가 품질 기준

### 6.1 AI 평가 시 준수 원칙 (평가 가이드 기반)

1. **과정 중심 평가**: 결과가 아닌 탐구 과정(접근 방식, 사고 과정, 변화)을 중심으로 평가
2. **구체성 기반 판단**: "심도 있게 탐구함" 같은 추상적 표현보다 실제 내용의 구체성으로 판단
3. **맥락적 성적 해석**: 등급을 그대로 해석하지 않고, 원점수-평균-표준편차-수강자수 종합 분석
4. **연결성 중시**: 개별 활동이 아닌 활동 간 연결, 학년 간 심화를 종합 판단
5. **학년별 차등 기대**: 1학년은 탐색기로 관대하게, 2학년은 구체화, 3학년은 심화 기대
6. **형식적 표현 감별**: "적극적으로 참여함", "성실하게 임함" 등 반복 표현의 실질 의미 판별
7. **진정성 판단**: 과장/형식적 기록 vs 구체적 행동/사고가 드러나는 기록 구분

### 6.2 예시 분석본에서 도출한 평가 패턴

**긍정 평가 패턴**:

- 탐구 계기가 명확하고 주제가 구체적인 경우
- 탐구 결과가 수치화되거나 구체적 결론이 도출된 경우
- 학년 간/과목 간 탐구 주제의 연속성이 있는 경우
- 비판적 사고가 드러나는 경우 (양면적 분석, 한계 인식)
- 주도적으로 활동을 기획하거나 리더 역할을 한 경우
- 다른 과목에서 같은 역량이 반복 확인되어 신뢰도가 높아지는 경우

**부정 평가 패턴**:

- "보고서를 작성함", "탐구를 진행함"으로 끝나고 구체적 내용 없음
- "흥미를 느낌", "유익했음", "의지를 밝힘"으로 마무리되는 추상적 결론
- 여러 활동의 나열 without 깊이 (1500바이트에 3개 이상 활동)
- 진로와 무관한 활동 과다
- "추가 탐구 의지 표명"만 하고 실제 추가 탐구가 없는 패턴 반복
- 학습 목표의 단순 나열 ("~를 이해함", "~를 설명함")
- **과목다운 세특 부재**: 과목 본질과 무관한 소재만 활용 (예: 물리학 세특인데 자동차 역사 조사)
- **주제 반복 without 심화**: 같은 키워드가 여러 과목에서 반복되지만 깊이가 동일 수준

**활동 밀도 분석 기준**:

- 세특 분량 대비 활동 수를 확인하여 깊이 평가
- 1500바이트(약 500자) 내 활동 1~2개 = 적정 밀도 (깊이 있는 서술 가능)
- 1500바이트 내 활동 3개 이상 = 나열식 (각 활동의 깊이 부족)
- 적은 활동이라도 탐구 구조(문제 설정->과정->결과->확장)가 갖춰진 서술을 높이 평가

## 7. 조건부 섹션 처리

### 7.1 대학 추천 (입시 전략 섹션)

- **고2 이상 생기부 포함**: `admissionStrategy` 제공 (대학 추천 + 입시 전략 전체)
- **고1 생기부만**: `admissionStrategy` 대신 `directionGuide` 제공 (어떤 계열/학과 방향이 맞는지, 2학년 때 어떤 과목을 선택하면 좋을지). **2028학년도 계열별 권장과목 데이터를 활용**하여 목표 계열에 따른 2학년 권장 선택과목을 구체적으로 안내하고, 메디컬 지망 시 대학별 필수 과목을 사전 안내 (참조: `libs/report/constants/recommended-courses.ts`)

### 7.2 모의고사 데이터

- **모의고사 데이터 있음** (`hasMockExamData: true`): 정시 전략 포함, 수능 최저 전략 분석
- **모의고사 데이터 없음**: 내신 기반 전략만 제공, 정시 전략은 "데이터 부족으로 분석 불가" 표시

### 7.3 목표 학과 미정

- **목표 학과 있음**: 전공 관련 교과 이수 노력/성취도 분석, courseAlignment 권장과목 매칭
- **목표 학과 없음**: 생기부에서 드러나는 관심 분야 기반 학과 추천 (majorExploration 활용) + 핵심 권장과목 안내

### 7.4 교육과정 버전

- **2022 개정 적용 학생** (2025학년도 고1부터): 창체 3영역(자율/자치, 동아리, 진로), 소인수 5명 이하
- **2015 개정 적용 학생** (2026학년도 고3): 창체 4영역(자율, 동아리, 봉사, 진로), 소인수 13명 이하
- `activityAnalysis`의 `curriculumVersion` 필드로 자동 전환

## 9. 용어 정의

| 용어           | 정의                                                                              |
| -------------- | --------------------------------------------------------------------------------- |
| 세특           | 세부능력 및 특기사항. 과목별 교사의 서술형 평가                                   |
| 창체           | 창의적 체험활동. 자율/자치, 동아리, 진로 활동 (2022 개정 기준)                    |
| 학종           | 학생부종합전형. 생기부 기반 대입 전형                                             |
| 교과           | 학생부교과전형. 내신 등급 기반 대입 전형                                          |
| 정시           | 정시모집. 수능 성적 기반 대입 전형                                                |
| 역량 태깅      | 세특/창체 서술에서 드러나는 평가 역량을 식별하여 태그 부여 (`CompetencyTag` 타입) |
| 역량 스코어    | 학업(100) + 진로(100) + 공동체(100) = 300점 만점 정량 점수                        |
| 꼬리물기 탐구  | A에서 출발 -> B에 대한 호기심 -> C로 심화하는 연쇄적 탐구 구조                    |
| 나열식 서술    | 여러 활동을 깊이 없이 단순 나열하는 세특 서술 패턴                                |
| 환산 등급      | 고교 유형(일반고/특목고/자사고/특성화고)에 따라 보정된 등급                       |
| 소인수 과목    | 수강자 5명 이하(2022 개정) 또는 13명 이하(2015 개정) 과목. 석차등급 미부여        |
| 레이더 차트    | 4대 역량(학업/진로/공동체/발전가능성)을 0~100 스케일로 시각화한 방사형 차트       |
| 유형명         | 학생의 핵심 특성을 2~4단어로 요약한 분류명 (예: "탐구형 성장러")                  |
| 캐치프레이즈   | 학생의 생기부 스토리를 한 문장으로 요약한 표현                                    |
| directionGuide | 고1 전용 섹션. admissionStrategy 대체, 진로 방향 설정 가이드                      |
| 활동 밀도      | 세특 분량(바이트) 대비 서술된 활동 수. 밀도가 높을수록 나열식 위험                |
